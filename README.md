**Telco Customer Churn Prediction**

**Problem Statement**
Predict whether a customer will churn using customer demographics and service usage data.

**Dataset**
Telco Customer Churn dataset from Kaggle.

**Models Used**
- Logistic Regression
- Random Forest Classifier

**Evaluation Metrics**
- Accuracy
- Recall
- F1-score

**Key Insights**
- Contract type and tenure strongly influence churn
- Threshold tuning improved churn recall
